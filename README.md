# sasharimsh.ru

Статический сайт психоаналитической практики Александры Римш.

## Технологии

- Чистый HTML / CSS / JavaScript
- Без сборщиков, без backend, без зависимостей
- Шрифты подключены через Google Fonts (Cormorant Garamond + Inter)
- Готов к развёртыванию на GitHub Pages, Cloudflare Pages, Netlify

## Что нужно добавить вручную

В папку `images/` положить:

- `hero-portrait.jpg` — главный портрет для hero-блока (рекомендация: 1200×1500px, формат JPG, оптимизация для веба)
- `og-cover.jpg` — обложка для соцсетей (1200×630px)
- `favicon.png` — иконка сайта (минимум 32×32px, лучше 512×512px)

Пока изображений нет, в hero-блоке отображается placeholder.

## Локальный запуск

Просто открыть `index.html` в браузере. Никакой сборки не нужно.

Для проверки относительных путей удобно запустить локальный сервер:

```bash
# Python 3
python3 -m http.server 8000

# Или Node.js
npx serve .
```

И открыть `http://localhost:8000`.

## Что НЕ настроено и почему

- **Аналитика (Яндекс.Метрика, Google Analytics)** — не подключена. Решение про аналитику принимается отдельно, исходя из позиционирования сайта.
- **Формы обратной связи на сайте** — нет. Связь идёт через Telegram/WhatsApp/email. Это сознательное решение: меньше трения, больше доверия.
- **Cookie-баннер** — не нужен, так как нет систем сбора данных. При подключении аналитики потребуется добавить.
- **API-ключи, токены, секреты** — отсутствуют по требованию ТЗ.
