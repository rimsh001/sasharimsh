# Папка images/

Сюда положить:

## hero-portrait.jpg
Главный портрет для hero-блока на главной странице.
- Рекомендуемый размер: 1200×1500px (соотношение 4:5)
- Формат: JPG (оптимизированный для веба, ~150-300KB)
- Содержание: профессиональный портрет, прямой взгляд, тёплое освещение, простой фон

## og-cover.jpg
Обложка для соцсетей (Open Graph).
- Размер: 1200×630px
- Формат: JPG
- Содержание: фотография + минимальный текст «Александра Римш · Психоаналитический подход»

## favicon.png
Иконка сайта в браузере.
- Размер: минимум 32×32, лучше 512×512
- Формат: PNG с прозрачностью
- Содержание: монограмма «АР» в фирменном цвете

## После добавления изображений

В `index.html` найти блок:

```html
<div class="hero__image">
  <div class="hero__image-placeholder">
    [&nbsp;портрет&nbsp;]<br>
    images/hero-portrait.jpg
  </div>
</div>
```

И заменить на:

```html
<div class="hero__image">
  <img src="images/hero-portrait.jpg" alt="Александра Римш" loading="eager">
</div>
```

Этот файл (README.md) можно удалить после загрузки изображений.
