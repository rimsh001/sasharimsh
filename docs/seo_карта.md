# SEO-карта sasharimsh.ru

## Стратегия

Сайт целится в **узкую качественную аудиторию**, не в массовый трафик. Цель не «миллион посетителей», а 30–100 квалифицированных в месяц через 6–12 месяцев.

Основной поисковый трафик ожидаем по двум типам запросов:

1. **Брендовые** — «Александра Римш», «sasharimsh» (когда появится узнаваемость)
2. **Тематические длинные запросы** — «психоаналитик в Москве с глубокой работой», «психотерапия привязанности», «почему понимание не освобождает» (по тексту эссе)

Мы НЕ конкурируем по высокочастотным запросам типа «психолог Москва» — там сидят агрегаторы и большие практики с миллионными бюджетами.

## Open Graph / соцсети

Все страницы имеют OG-разметку. Для соцсетей используется:
- `og:image` → `https://sasharimsh.ru/images/og-cover.jpg` (1200×630px)
- `og:locale` → `ru_RU`
- `og:type` → `website` для общих, `article` для эссе, `profile` для about

При публикации ссылок в Telegram, ВКонтакте, WhatsApp — будет красивая карточка с заголовком, описанием и обложкой.

## Технические настройки

### robots.txt
Все страницы открыты для индексации, кроме:
- `/404.html` — служебная

### sitemap.xml
Перечислены все 8 канонических URL с приоритетами:
- 1.0 — главная
- 0.9 — about, services, consultation
- 0.8 — articles, article-1
- 0.7 — contacts
- 0.3 — privacy

### Canonical
Каждая страница содержит `<link rel="canonical" href="...">` — это защищает от дублей при попадании в индекс с разными параметрами URL.

### noindex для приватных страниц
- `privacy.html` — `noindex, follow`
- `404.html` — `noindex, nofollow`

## Что нужно сделать после запуска

1. **Регистрация в поисковиках:**
   - Яндекс.Вебмастер: добавить сайт, подтвердить владение, отправить sitemap.xml
   - Google Search Console: то же самое

2. **Добавление в каталоги (опционально):**
   - Яндекс.Карты — карточка организации (если есть кабинет)
   - 2ГИС — то же

3. **Внешние ссылки:**
   - Профили в Telegram, ВКонтакте, профессиональные ассоциации — со ссылками на сайт
   - Гостевые публикации в коллегиальных каналах — ссылка обратно

4. **Контент-стратегия:**
   - Раз в 1–2 месяца — новый авторский эссе на сайте (новый article-N.html)
   - Постоянный поток в Telegram-канал — ссылки на сайт там, где уместно

## Чего НЕ делать

- Не покупать seo-ссылки на биржах (вредно для авторитетных доменов)
- Не использовать техники «спамдекс»: переоптимизация ключей, скрытый текст
- Не публиковать «псевдоэкспертный» контент массового типа («10 признаков...», «как избавиться от...») — это противоречит позиционированию
- Не делать «облако тегов» в подвале и другие маркеры старых SEO-практик

## Метрика успеха

Через 3 месяца после запуска:
- 5–15 уникальных посетителей в день
- 3–5 органических переходов из поисковиков в день
- Топ-10 в Яндексе и Google по брендовому запросу «Александра Римш»

Через 6 месяцев:
- 15–40 посетителей в день
- 10–20 органических переходов в день
- Несколько внешних ссылок на article-1.html (если эссе действительно сильное и его шерят)

Через 12 месяцев:
- 40–100 посетителей в день
- Стабильный SEO-трафик по длинным хвостам
- Несколько публикаций в крупных СМИ и каналах со ссылкой на сайт
